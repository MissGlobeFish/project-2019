### 目录结构

* doc 文档
* js js代码
  * common 组件
  * config 配置文件
  * navigations 路由
  * page 完整页面
    * /page/ConsoleStation 控制台
    * /page/Comprehensive 综合
    * /page/My 我的
    * /page/Home 首页
  * redux 状态管理
  * util 工具类
* res 静态资源
* App.js 入口
